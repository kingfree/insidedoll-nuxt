## 📝 快速开始

### 立即开始翻译核心页面

```bash
# 创建核心页面翻译模板
npm run translation:create \
  index.md \
  main.md \
  info.md \
  wnew.md

# 查看待翻译文件
npm run translation:list 10

# 继续 storyg1 系列
npm run translation:create storyg1/0g/story_0g33.md

# 查看进度
npm run translation:report
```

### 完成 storyg1 系列

```bash
# 批量创建剩余文件模板
npm run translation:create \
  storyg1/0g/story_0g33.md \
  storyg1/0g/story_0g34.md \
  storyg1/0g/story_0g35.md
  # ... 继续添加
```

### 开始新系列

```bash
# 如果选择 storyh1（最小系列）
npm run translation:create storyh1/h0/story_h001.md

# 如果选择 story01（主力系列）
npm run translation:create story01/01/story_0101.md
```

## 🎨 内容类型说明

### 小说故事
- 主要是美少女着装主题的小说
- 包含成人内容，需要注意翻译尺度
- 保持原文的叙事风格和氛围

### 访谈问答
- Q&A格式的互动内容
- 相对简短，但数量多
- 可以作为休息时的轻松翻译内容

### 博客文章
- 2010-2012年的个人博客
- 时效性内容，优先级较低
- 可以了解作者创作背景

## 📈 翻译进度跟踪

使用工具查看实时进度：

```bash
# 完整报告
npm run translation:report

# 只看待翻译列表
npm run translation:list

# 导航结构
npm run nav:analyze
```

**提示**：根据个人兴趣和时间安排选择翻译路径，不必严格按照顺序进行。
